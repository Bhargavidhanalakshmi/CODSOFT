CodeSoft_ToDoList
  
Description

A simple text-based To-Do List application developed in Python for the CodeSoft Python Internship.

Features

. Add new tasks . View all tasks . Mark tasks as completed . Delete tasks
. Menu-driven interface
. Easy to use

 Concepts Used

. Python Lists
. Functions
. Loops
. If-Else Statements
. User Input
  
Technologies Used
. Python 3

 How to Run
 Run the following command 
 python "to do list.py"
