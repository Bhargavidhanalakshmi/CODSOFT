# CodeSoft_ToDoList

## Description
A simple text-based To-Do List application developed in Python for the CodeSoft Python Internship.

## Features
1. Add new tasks
2. View all tasks
3. Mark tasks as completed
4. Delete tasks

## Concepts Used
- Python Lists
- Functions
- Loops
- If-Else Statements
- User Input

## Technologies Used
- Python 3

## How to Run
python "to do list.py"
